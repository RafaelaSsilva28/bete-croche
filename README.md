# 🧶 Ateliê Elizabete - API

API REST desenvolvida para o sistema do *Ateliê Elizabete*, responsável pelo gerenciamento de produtos, categorias, usuários, autenticação e upload de imagens do site.

## 📌 Objetivo

Fornecer uma API segura e organizada para administrar o catálogo de produtos do Ateliê Elizabete, permitindo o gerenciamento completo das informações exibidas no site.

## 🚀 Funcionalidades

- 🔐 Autenticação de usuários
- 👤 Gerenciamento de usuários
- 📦 Cadastro, edição e exclusão de produtos
- 🏷️ Gerenciamento de categorias
- 🖼️ Upload de imagens dos produtos
- 📖 Documentação da API com Swagger
- 🗄️ Integração com banco de dados SQL

## 🛠️ Tecnologias

- Node.js
- Express.js
- PostgreSQL
- Swagger
- JWT (JSON Web Token)
- Multer
- JavaScript

## ▶️ Executando o projeto

1. Clone o repositório

bash
git clone https://github.com/RafaelaSsilva28/bete-croche.git


2. Instale as dependências

bash
npm install


3. Configure as variáveis de ambiente.

4. Inicie a aplicação

bash
npm start


ou

bash
npm run dev


## 📚 Documentação

A documentação da API está disponível através do *Swagger*, permitindo visualizar e testar todos os endpoints da aplicação.
